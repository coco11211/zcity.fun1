About Z-City

Server Z-City

Z-City is a server in the game Garry's Mod, with such modes as:

Homicide TDM And more...

Features

The server features realistic body behavior, bleeding, ragdoll control, and a well-thought-out weapon system.

Realistic Physics
Authentic body behavior and ragdoll control
Bleeding System
Dynamic bleeding mechanics
Weapon System
Well-thought-out weaponry
Slower Gameplay
Strategic and tactical pace

Evolution from Homigrad

Compared to its previous project, Homigrad, Z-City differs both visually and mechanically. Gameplay on the server is also significantly slower than in Homigrad, offering a more tactical and immersive experience.
